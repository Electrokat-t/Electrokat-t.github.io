How to add folder or directory in github repository?

If you are a beginner or have just started to use Githib. You may know how to create repository but not how to add folders in it. First, let's create a new repository or you can create a folder in existing one. In repository, if you click on (•••) you can see the option to create a new file or to upload files but not for adding folders.

To create a folder or directory in Github:

  1. Select option to create a file [Screen shot area]
  2. Now type the name of the folder, which you want to create. Don't worry we are creating folder here not file. [Screen shot]
  3. After typing the name put a (/) after it. It will create a folder for us.
That's it! now type the name of file you want to create inside. Remember you can't create a folder with a blank file in it. So make sure to type something in it.
